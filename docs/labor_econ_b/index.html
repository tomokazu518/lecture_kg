  • なお，提出すれば回答を保存していなくても成績に反映されます。

第3回　労働力調査についての課題

問題

労働力調査(2024年2月)を調べて答えよ。

  1. 【原数値】　15歳以上人口　（万人単位の整数で答えよ）
  2. 【原数値】　労働力人口　（万人単位の整数で答えよ）
  3. 【原数値】　雇用者にしめる非正規の職員・従業員の割合（男性）　　（パーセントで小数点以下1桁で答えよ）
  4. 【原数値】　雇用者にしめる非正規の職員・従業員の割合（女性）　　（パーセントで小数点以下1桁で答えよ）
  5. 【季節調整値】　完全失業率　　（パーセントで小数点以下1桁で答えよ）

15歳以上人口のうち，就業者は6825万人，完全失業者が175万人，非労働力人口が4000万人いるとして，問いに答えよ。

【注意】

  • 計算式などは書かず，答えのみ入力すること
  • 答えはパーセントで，割り切れない場合には，小数点以下第二位を四捨五入し，少数第一位まで答えよ
  • 単位（％やパーセント）はつけず数値のみ答えること
  • 自動採点なのでイレギュラーな回答は正解になりません
  1. 労働力率(労働力人口比率)は何パーセントか？
  2. 完全失業率は何パーセントか？
  3. いま完全失業者のうち35万人が職に就いたとすれば，完全失業率は何パーセントとなるか？
  4. 上の問いからさらに，完全失業者のうち40万人が職探しをやめたとすれば，労働力率は何パーセントになるか？
  5. このとき，完全失業率は何パーセントとなるか？

  2. 6905
  3. 23.1

  5. 2.6
  6. 63.6
  7. 2.5
  8. 2.0

解説 1~5

総務省統計局のホームページで数字を調べるだけです。統計データを調べるということも重要なスキルです。

解説 6~10

まず15歳以上人口は，就業者＋完全失業者＋非労働力人口＝11000万人です。そのうち，労働力人口は就業者＋完全失業者＝7000万人なので，労働力率＝7000万人/11000万人×100＝63.6パーセントとなります。完全失業率は完全失業者数/労働力人口なので，175万人/7000万人×100＝2.5パーセントです。

完全失業者のうち，35万人が職に就いたとすれば，完全失業者数は140万人になります。労働力人口は変わらないので7000万人です。したがって，完全失業率は140万人/7000万人＝2パーセントです。

さらに完全失業者のうち40万人が職探しをやめたとすれば，職探しをやめた40万人は非労働力人口になります。したがって，労働力人口は40万人減少して6960万人となり，完全失業者数は100万人になります。労働力率は，6960万人/11000万人＝63.3パーセント，完全失業率は100万人/6960万人＝1.4パーセントです。

重要なことは，完全失業者が仕事を見つけた場合だけではなく，職探しをやめてしまう場合でも失業率が低下するということです。

第5回　大学教育の収益率についての課題

出題にいくつか瑕疵がありました。私的費用・便益について答えるということも明示していなかったので，追加で課題を出しました。

大学教育の費用と便益については，講義資料の図で確認してください。

  • 大学無償化･･･当然，私的費用が下がります
  • 景気の悪化･･･景気の悪化は，賃金や雇用に影響を与えます。大学に進学することの機会費用は，在学中に働いていれば稼げる所得なので,賃金の低下は大学教育の機会費用を低下させます。とくに，高校卒の方が景気の悪化の影響を受けやすいということは，労働市場の瑕疵効果のところで説明しました
  • 定年延長･･･大学を卒業した場合に，高い賃金で働くことができる期間が長くなるということなので，大学教育の便益を増加させます
  • 保育所の整備による定員増加･･･子育てによるキャリアの中断を短くすることができるため，高い賃金で働くことができる期間が長くなり，大学教育の便益を増加させます

7番目の問題については，正誤を明確に判断できないものがあったので採点していません(配点をゼロにしてほかの問題に振り分けました)。

第5回 (大学教育の収益率)の追加課題

  • 問題
  • Forms

教育の費用と便益の求め方だけ解説しておきます。まず，高校卒労働者の賃金プロファイルは直線で，18歳のときの年収が240万円，60歳の時の年収が620万円なので，年収は1年に10万円ずつ上昇することがわかります (42年間で420万円上昇)。したがって，高校卒労働者の22歳のときの年収は240万円です。

大学教育の直接費用は，1年あたり250万円なので4年間では1000万円です。間接費用は，18〜22歳の4年間における高校卒労働者の賃金収入なので，(200万円＋240万円)×4年/2＝880万円です。台形の面積が平行な2辺の合計×高さ/2で求められることを思い出してください。便益は22〜60歳における高校卒と大学卒の賃金格差の総額なので，(60万円＋250万円)×38年/2＝5890万円です。
